# ClawHunter - Your Agentic Career Partner

## Overview
ClawHunter is an AI-powered job matching platform that uses your LinkedIn profile to find high-impact roles in top tech companies.

## Features
- Google-like search interface for LinkedIn profiles.
- Intelligent matching based on location, title, and company prestige.
- AI-generated match reasoning and expected salary ranges.
- Automated email alerts for new matches.

## Local Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`

## Deployment
Deployed on Railway via GitHub integration.

## Configuration
Configure LLM API variables in the Railway Environment settings.
